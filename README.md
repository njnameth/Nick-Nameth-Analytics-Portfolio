The MSBA Financials Hypothetical Portfolio Analysis project is an AWS-based data pipeline designed by Nick Nameth to streamline the identification of potential company acquisitions. The system addresses the challenges of managing unstructured data from multiple sources and replacing inconsistent data flow jobs with a robust, automated architecture.

The workflow begins with data ingestion, where raw CSV files, including financial records, ratio data, and bankruptcy history,are stored in Amazon S3 buckets. These datasets are then processed through an AWS Glue ETL job that standardizes schemas and joins the information based on a unique client ID. The transformed data is subsequently loaded into an Amazon Redshift data warehouse, allowing for complex SQL analysis and the creation of a master data sheet. The pipeline concludes with predictive modeling in Amazon SageMaker, which evaluates the consolidated data to generate bankruptcy predictions and company profiles, ultimately providing actionable insights for investment and acquisition decision-making.
